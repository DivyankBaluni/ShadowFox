## 🗂️ Projects

### 01 · Personal Portfolio Website

> A personal portfolio website built as the first task of the internship.

| Detail | Info |
|--------|------|
| 🔗 **Live Demo** | [portfolio-shadowfx.netlify.app](https://portfolio-shadowfx.netlify.app/) |
| 🛠️ **Tech Stack** | HTML, CSS, JavaScript |
| 📁 **Folder** | `Portfolio/` |

**About the project:**
A responsive personal portfolio website showcasing my background, skills, and projects. Built from scratch using only vanilla HTML, CSS, and JavaScript — no frameworks or libraries.

**Key Features:**
- Responsive layout that works on desktop and mobile
- Sections for About, Skills, Projects, and Contact
- Smooth scroll navigation
- Clean, minimal design
